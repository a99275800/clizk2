## 18f77m testando outliners like workflowy

9 jul 2020: testei:

**hypernote.io**: sofisticado e cheio de manhas, mas incompleto: nao conhece markdown, não dá para mover os nodes, não exporta

**checkvist**: bem bom mesmo, mas muito caro (embora dê para fazer quase tudo na versão grátis); só que é feio e não dá para customizar; faz-se tudo usando teclado! E formata markdown inclusive reconhecendo notas de rodapé! A versão paga é mais cara que o Scrivener e o Notion.

[[32d63y\|32d63y]] checkvist personalização dicas e truques

**workflowy**: bonzinho, dá para fazer muito com a versão grátis, menos sofisticado que o checkvist; esse dá para exportar para o Notion; e tem uma versão desktop

**remnote** um excelente apróprio notion like, não é um outliner, é um zettelkasten mesmo, que transforma os "átomos" de texto (parágrafos) em flashcards  para testes e faz contabilidade de acertos, repetição programada, revisão agendada, etc. muito bom para concurso, recomendei para Vitor e meninas do gabinete; é pago mas a versão grátis basta; v. senha.
