## estética, index

> cvç: RQR 6N2 e tag #estética 

> Para não confundir os verbetes / tags *estética*, *narratologia*  e *arte* : (a) **estética**, filosofia, estudo dos conceitos de belo (beleza) e sublime; (b) **arte**: para dizer o que é arte, para que serve arte, como se reconhece uma obra de arte; (c) **narratologia**: aquela parte técnica sobre como interpretar / analisar as peças artísticas, achar seu significado ou sentido e tal;
