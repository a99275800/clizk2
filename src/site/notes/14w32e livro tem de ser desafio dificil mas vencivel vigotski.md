Algo difícil pode vir a se tornar mais simples, como afirma [Lev] Vigotsky, pedagogo, com sua “zona de desenvolvimento proximal”. Para educar verdadeiramente, é preciso que o educador apresente problemas minimamente conhecidos pelos estudantes, mas com certa carga de desafio. Uma vez que o estudante processe aquele problema, uma nova zona será criada com um desafio maior

Para ler com verdade é preciso ter humildade e vontade de inauguração, de conhecer as palavras e as coisas como se pela primeira vez, quase como uma criança
