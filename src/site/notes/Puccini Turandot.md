## Puccini, Turandot

----

 _vb. criado em 17/02/2014, 17h13m._

Fria como a espada é a bela Turandot!

Aqui todos os cemitérios estão ocupados! / Aqui bastam os loucos nativos! / Não queremos mais loucos forasteiros!

o enterro / Para ti se prepara! / ... / Por uma princesa! / Que coisa é? / Uma mulher com corôa na cabeça... / ...e o manto com galões! / Mas se a desnuda... / É carne! / É carne crua! / É coisa... / ...que não se come! / Deixe a mulher! / Ou tome cem esposas, / Que, no fundo, a mais sublime / Turandot do mundo / Tem uma face, dois braços, / E duas pernas, sim, bela, imperial, / Sim, sim, bela, sim, mas sempre ela! / Com cem esposas, Ó tolo, / Terás pernas em abundância, / Duzentos braços, / E cem meigos peitos. / Cem peitos... / ...Espalhados por cem leitos...

Noite sem luz... / ...Garganta negra de um caminho... / ...São mais claras que os enigmas de Turandot. / Ferro, bronze, muro, rocha... /

...A obstinada tua cabeça... / ...São menos duros que os enigmas de Turandot.

Tu passas sobre um pobre coração

Tudo andava segundo / A antiquíssima regra do mundo. / E então nasceu... / Turandot!

A que estamos reduzidos? / Somos os ministros do carrasco!

Tenho uma casa em Honan / Com o seu laguinho azul, / Todo cercado de bambu. / E estou aqui a dissipar a minha vida, / A cansar o cérebro em cima de livros / Sagrados

Oh! mundo cheio de loucos enamorados!

Nós sonhamos e o palácio já formiga / De lanternas, de criados e de soldados. / Ouve-se o grande tambor / Do templo verde! / Já se ouvem os incontáveis / Chinelos de Pequim.

Un giuramento atroce mi costringe / a tener fede al fosco patto

o cetro sagrado que empunho / goteja sangue.

Mas quem enfrente a prova / E vencido fique / Dará ao machado a soberba / cabeça!

A esperança:

 Na noite escura voa um fantasma / Iluminado. / Sobe e abre as asas / Sobre a negra infinita humanidade. / Todos o invocam / E todos lhe imploram. / Mas o fantasma desaparece com a Aurora para renascer no coração. / E cada noite nasce e cada dia morre!

Esta noite ninguém durma em Pequim! (Questa notte nessun dorma in Pekino!)

Inútil súplica! Inútil ameaça! / Pereça o mundo, quero Turandot! (Crollasse il mondo, / voglio Turandot!)
