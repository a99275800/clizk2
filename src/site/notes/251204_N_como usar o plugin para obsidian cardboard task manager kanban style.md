So, if you had the following in one of your markdown files:

```
<del>- [x] run erands @due(2022-12-02) @completed(2025-12-04T22:10:28-03:00)</del>
  - [x] do shopping 
  - [x] wash car  @completed(2025-12-04T22:11:08-03:00)
  - [x] cook dinner @completed(2025-12-04T22:11:09-03:00)

  perhaps I should look up some [[example_tasks/recipes|recipes]] first

  - [x] do something with a long title that will truncate when displayed @completed(2025-12-04T22:11:09-03:00)
  - [x] go to bed @completed(2025-12-04T22:11:10-03:00)
```
- Use one of the commonmark supported unordered list formats:
    - `- [ ] Task title`
    - `* [ ] Task title`
    - `+ [ ] Task title`
