## 78y92q outras teclas de atalho - não zettelkasten - para o vscode

> v. também [[15r59k\|15r59k]] teclas de atalho personalizadas para zettelkasten no vscode

### para trabalhar com tela dividida

dividir a tela: Ctrl + ]

alternar para o "grupo" da esquerda: Ctrl + 1

alternar para o "grupo" da direita: Ctrl + 2

alterna comenta/descomenta a linha inteira: Ctrl + ;

alterna comenta/descomenta o texto selecionado: Alt + Shift + A

mostra lista dos files abertos: `Ctrl+K Ctrl+P`
