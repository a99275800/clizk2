Falta acertar o troço Lintalist... Por causa das tags...

Mas como o vscode, com as extensões próprias para Markdown dadas pela extensão "Foam for VSCode", preenche as tags (autocompleta) dinamicamente, e também cria links para outros verbetes usando atalhos simples, talvez o Lintalist seja supérfluo agora.
