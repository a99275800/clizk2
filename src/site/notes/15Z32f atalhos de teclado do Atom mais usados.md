## 20191019092217 atalhos de teclado do Atom mais usados

`ctrl + .` --> para descobrir para q serve um atalho

`ctrl + f` --> pesquisa no arquivo atual

`ctrl + shift + f` --> pesquisa em todos os files do projeto (n esqueça de recarregar o projeto qdo criar um file novo, ele nao adiciona automatica/); v. tb [[20191018093828\|20191018093828]] pesquisa em Atom usaG regex

`ctrl+\`--> abrir a árvore de files

`ctrl+clique`--> cursor múltiplo para selecionar múltiplas posições do file, daí digita uma vez e ele digita em tudo
