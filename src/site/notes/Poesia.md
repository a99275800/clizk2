1. Alguns gostam de poesia // Alguns - ou seja nem todos. // Nem mesmo a maioria de todos, mas a minoria. // Sem contar a escola onde é obrigatório e os próprios poetas seriam talvez uns dois em mil. // Gostam - // mas também se gosta de canja de galinha, gosta-se de galanteios e da cor azul, gosta-se de um xale velho, gosta-se de fazer o que se tem vontade gosta-se de afagar um cão. // De poesia - // mas o que é isso, poesia. // Muita resposta vaga já foi dada a essa pergunta. // Pois eu não sei e não sei e me agarro a isso como a uma tábua de salvação.
