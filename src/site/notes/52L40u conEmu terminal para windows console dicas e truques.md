## 52L40u conEmu terminal para windows console dicas e truques

para sair do modo grudado sem menus nem barra de título, Win+Alt+P

para mostrar e esconder o terminal usa Ctrl+shift+space

Win+S cria novo terminal (devia duplicar-clonar mas nao funfa; mas copia o histórico do original para o novo)

para alternar, ctrl+tab

apps+F abre gui para pesquisar na tela (no texto que está ou esteve na tela); apps é a tecla que substitui o botão dto do mouse
