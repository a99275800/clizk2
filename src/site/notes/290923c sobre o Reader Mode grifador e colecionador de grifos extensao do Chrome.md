Reader Mode Pro criei conta

é uma porcaria de extensão para Chrome, para substiuir o Mercury Reader. É um pobre subsituto para Diigo e Glasp. Mas já paguei por uma licença perpétua US(cifrão) 15.

Vantagens: permite grifar, e salvar a versão grifada em PDF; é bem customizável de aparência; dá para copiar os grifados em formato texto; o pegador de biblio funciona nele; entende as páginas traduzidas pelo Chrome;

Defeitos: não tem um painel como o do Diigo com título e tags e grifos, no painel bosta deles só tem o link; se achar o link e tiver ainda a extensão, consegue ver a versão grifada; não exporta em nenhum outro formato, só PDF

Tem de instalar a extensão no Chrome e na primeira execução colocar a License Key; 2 PCs simultâneos no máximo. Para acessar o painel dashboard, o único jeito é abrir um artigo para grifar e clicar no botão de salvar, ou ir ao rodapé. Não tem link. Se precisar de senha, veja EV
