Las 6 reglas de George Orwell para escribir. 1 ﻿﻿﻿Nunca uses una metáfora, símil u otra frase hecha que estés acostumbrado a ver por escrito. 2) ﻿﻿﻿Nunca uses una palabra larga si puedes usar una corta que signifique lo mismo. 3) ﻿﻿﻿Si es posible eliminar una palabra, hazlo siempre. 4) ﻿﻿﻿Nunca uses la voz pasiva cuando puedas usar la activa. 5) ﻿﻿﻿Nunca uses una expresión extranjera, una palabra científica o un término de jerga si puedes pensar en una palabra equivalente en tu idioma que sea de uso común; 6)  ﻿﻿﻿Incumple cualquier regla antes de escribir nada estúpido.
