## 20190403105050 ortellado extrema direita quer redefinir o campo político

nexo, [[Pablo Ortellado\|Pablo Ortellado]], ev 110119b; a extrema direita quer se tornar o novo centro, o novo "normal"; bolsonaro dá enraizamento a um projeto de redefinição do espectro político (que ameaça valores democráticos) onde os ultraconservadores afirmam a reorganização do  espectro políico (redefinir o campo político) para se tornarem o novo centro, empurrando para as margens o *old stablishment*; bolsonaro  usa sua grande votação para tentar fazer força gravitacional para fazer todo mundo políico girar em torno das novas definições que propõe;[^a468];
