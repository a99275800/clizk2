Let's Get the Game Working!

You want a fast and easy way to get this game running, you came to the right place.  
  
Ok to cut a long story short, this game came out a number of years ago (10+) and wasn't designed with current hardware/graphical advances in mind, namely being multiple cores and threading. So a large percentage of those who initially run SimCity 4 may experience problems. SimCity 4 is still the best city building game there is, and the number 1 reason it is voted down on steam is because of technical issues. Let's fix this!  
  
This guide is intended to get this game working for you as fast and easily as possible. I am not saying this is the best method for results, if you delve deeper into websites like Simtropolis, you may find an "external launcher" will grant you better performance. However, such launchers can be tricky/fiddly to setup- this guide is a hassle free way to get this game up and running.  
  
  

Step 1: Make it Compatible with Windows XP

Assuming you are running the steam version of Sim City 4, go into your steam folder and access your SimCity 4.exe ; located at C:\Program Files (x86)\Steam\steamapps\common\SimCity 4 Deluxe/Apps.  
Right click on the icon, go to Properties, select Compatibility, make sure windows XP is selected in compatibility and the box is ticked for running the game in this mode.  
  

Step 2: Adjust your Steam Launcher

On your steam account go to your Library page where all your games are kept. Down the left hand you should see SimCity 4, right click, select Properties, click on the Launch Options button; in this paste in the command  

-CustomResolution:enabled -r1920x1080x32 -intro:off -CPUCount:1

  

Step 3: Adjust the in game options to remove the graphic issues

In the game, select the Options icon that is located at the bottom of the screen. Then select Graphic Options, go to Renderer and select Software, make sure shadows is also set to low.  
Now restart SimCity 4.

For those having trouble launching the game:  
-Right click the game in your library  
-Properties  
-Paste the following into your launch options without the quotation marks  
"-d:soft -intro:off -CPUCount:1 -f -CustomResolution:enabled -r1920x1080x32"  
  
If you're still having trouble, do this AND run compatibility for Windows 8.
