2. [[RQR rol index grandes temas ou questoes recorrentes fichas principais\|RQR rol index grandes temas ou questoes recorrentes fichas principais]]- É o que temos hoje de mais parecido com um sumário, índice superior ou listagem de conteúdos. Foi originalmente criado para ser uma tabela de correspondência entre tags/verbetes do OZ com seus equivalentes no fichário de papel Z. Contém as tags nativas do OZ, ou seja, preparadas para o Obsidian ou qualquer outra versão do Z que use arquivos de texto puro (são evolução dos antigos RQR Rol de Questões Recorrentes, uma lista dos temas constantemente estudados.) Só as notas mais importantes atualmente usam isso, mas seria bom que todas usassem, para compatibilidade reversa. As tags do Evernote são em menor número e os RQRs listados ali (que fazem referência a fichas de papel do antigo Z) servem como indexes dos temas mais recorrentes, e também conectam as versões digitais do Z com a versão física, de onde vem o RQR. Na prática, se não desse tanto trabalho, esse file seria o índice geral do Z. Problema: o RQR serve para identificar um parágrafo dentro de um verbete; a tag só refere ao verbete inteiro, logo não ajuda a indicar partes de um todo (como parágrafos de uma ficha de leitura longa). (q.v., por exemplo, o que fizemos em [DICK, Sonhos Elétricos (no Evernote)](evernote:///view/124106736/s672/7bb25ba9-f9fe-7ea7-12f4-844b18151613/106f008d-bf72-4e7d-b93f-4a39a1910296) e também [[DICK, Sonhos Elétricos, fdl\|DICK, Sonhos Elétricos, fdl]], aqui).

Já se chamou "20191019124308 cat_obras index 3z - fixas feitas direto m SZ q n t cópia m papel m Z" e também "82N41m cat_obras index 3z - fixas feitas direto m SZ q n t copia m papel m Z".
