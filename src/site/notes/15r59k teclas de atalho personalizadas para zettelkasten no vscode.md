## 15r59k teclas de atalho personalizadas para zettelkasten no vscode

> v. também [[78y92q\|78y92q]] outras teclas de atalho - não zettelkasten - para o vscode

Ctrl + F3 and F = fold section

Ctrl + F3 and A = fold all sections

Ctrl + F3 and U = unfold section

Ctrl + F3 and L = unfold all sections

Ctrl + J        = terminal (toggle)

Ctrl + Alt + B  = barra lateral (toggle)

Ctrl + Shift + . = que focaliza os breadcrumbs, que servem como outline### Genial!!!

Ctrl + Shift + q = foco na lista (pasta) de arquivos do folder aberto (sidebar esq.)

mostra lista dos files abertos: `Ctrl+K Ctrl+P`

`Ctrl+Alt+Windows+N` criar novo file
